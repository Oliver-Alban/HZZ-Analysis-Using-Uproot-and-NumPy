This exercise is aimed at undergraduates interested in high-energy physics who want to learn how to program in Python. Provided are three exercises utilising ATLAS Open Data to rediscover the Higgs boson. Students should choose to work through either UprootHiggsBosonExercise.pdf or HZZNdarrayAnalysis.ipynb. The pdf challenges learners to use the information provided to create a Jupyter Notebook of their own. The provided notebook contains all the same information as well as the exact code needed to complete each activity.
